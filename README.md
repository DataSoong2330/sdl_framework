## Installation der SDL oder eines Moduls unter Linux

- ./configure
- make
- sudo make install

## Linker-Flags

- -lSDL2
- -lSDL2_gfx
- -lSDL2_image
- -lSDL2_mixer
- -lSDL2_net
- -lSDL2_ttf
